IT Infrastructure Blueprint

for a Startup with 20 Employees

1. Project Overview

This virtual IT infrastructure test environment simulates a complete startup network setup using Windows Server 2022, Active Directory, DNS/DHCP, VLANs, and Group Policies. It is built on a Mac M1 using Parallels Desktop to demonstrate technical understanding and practical implementation.

2. Network & Infrastructure

  • Router: Ubiquiti UniFi Dream Machine Pro
  • Switch: Netgear Smart Managed Switch (PoE-enabled)
  • Access Points: Ubiquiti UniFi 6 Lite (2–3 units)
  • Server: Windows Server 2022 (AD, DNS, DHCP, File Services)
  • NAS: Synology DS920+ for backup and file storage
  • UPS: Optional for power redundancy

Network Structure

  • VLAN 10: Office LAN (workstations)
  • VLAN 20: Servers
  • VLAN 30: VoIP and printers
  • VLAN 40: Guest Wi-Fi (isolated internet-only)
  • WLAN SSIDs: "Startup-Office" (internal) & "Startup-Guest" (guest access)

3. Client Management

  • OS: Windows 11 Pro, domain-joined
  • Standard Software: Microsoft 365, antivirus, PDF reader
  • Security: BitLocker encryption, automatic updates via WSUS/Intune
  • Software deployment via GPO or Microsoft Intune

4. Access & Permissions

Users and groups are centrally managed via Active Directory. The organizational structure is reflected in Organizational Units (OUs) such as Sales, Marketing, IT, and Management. Each department has its own group policies and access rules.

OU Structure Example:

  • OU=Company
  • ├── OU=Sales
  • ├── OU=Marketing
  • ├── OU=IT
  • └── OU=Management

Permission Management:

  • Shared folders mapped via GPO per department
  • Access based on security groups (e.g., SG_Sales_Read, SG_IT_Admin)
  • GPOs for desktop restrictions, software policies, login scripts

5. Backup & Cybersecurity

  • Backup strategy: 3-2-1 rule (3 copies, 2 media types, 1 off-site)
  • Use of Synology NAS + encrypted cloud storage (e.g., Wasabi, Azure)
  • Automated daily and weekly backup schedules
  • Firewall with DPI, intrusion detection, and threat alerts
  • Company-wide MFA, strong password policies, regular security training
  • BitLocker for all endpoints, secure BIOS/UEFI settings

6. Support & Monitoring

  • IT helpdesk: GLPI or Spiceworks for ticket management
  • Remote access with AnyDesk or Microsoft Remote Desktop
  • Monitoring: PRTG Network Monitor for network/server performance
  • Log collection: Windows Event Viewer + centralized syslog
  • Automated alerts for CPU, disk, memory, and uptime issues

7. Demo Environment

A fully functional demo lab has been set up on a Mac M1 using Parallels Desktop with Windows Server 2022. It includes:

  • AD DS installation and domain configuration
  • Sample OUs and user accounts
  • GPOs for desktop wallpaper and password policy
  • Optional: Windows 11 VM joined to the domain
