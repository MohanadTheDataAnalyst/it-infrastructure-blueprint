IT Infrastructure Blueprint

for a Startup with 20 Employees

1. Project Overview

This virtual IT infrastructure test environment simulates a complete startup network setup using Windows Server 2022, Active Directory, DNS/DHCP, VLANs, and Group Policies. It is built on a Mac M1 using Parallels Desktop to demonstrate technical understanding and practical implementation.

2. Network & Infrastructure

The network infrastructure is designed to support high performance, security, and scalability. It consists of enterprise-grade components configured in a segmented VLAN structure to isolate traffic between users, services, and guests.

2.1 Recommended Hardware

  • Router & Firewall: Ubiquiti UniFi Dream Machine Pro – All-in-one gateway with integrated security, VPN, DPI, and cloud management.
  • Switches: Netgear Smart Managed Switches (24-port, PoE+) – VLAN-capable, Power over Ethernet for access points and IP phones.
  • Access Points: Ubiquiti UniFi 6 Lite (x3) – High-performance Wi-Fi 6 coverage across all office zones.
  • Server: One physical server with Windows Server 2022 – Provides Active Directory, DNS, DHCP, and file services.
  • Storage: Synology NAS (DS920+) – Used for local backups and shared file access with RAID configuration.
  • Power Backup: APC Smart-UPS – Protects critical hardware during power outages.
  • Patch Panel & Rack: For structured cabling and maintenance-friendly setup.

2.2 Network Structure & VLANs

To increase network efficiency and security, VLAN segmentation is implemented. Each type of traffic is separated logically at the switch level:

  • VLAN 10 – Office LAN: For all employee client devices (PCs, laptops).
  • VLAN 20 – Servers: Dedicated to Windows Server and NAS.
  • VLAN 30 – VoIP & Printers: Isolated network for printers and IP phones.
  • VLAN 40 – Guest Wi-Fi: Internet-only access for visitors, no internal resource access.

2.3 Wireless Network

  • SSID 1 – Startup-Office: WPA3-encrypted, connected to VLAN 10 for employees.
  • SSID 2 – Startup-Guest: Isolated VLAN 40 network with internet access only.
  • Controller-based wireless management via UniFi Controller for centralized monitoring and configuration.

2.4 IP Addressing & DHCP

  • IP schema follows private addressing (e.g., 192.168.10.0/24 for VLAN 10, etc.).
  • DHCP provided centrally via Windows Server 2022.
  • DNS internal resolution for domain and services; external DNS via router fallback.

3. Client Management

  • OS: Windows 11 Pro, domain-joined
  • Standard Software: Microsoft 365, antivirus, PDF reader
  • Security: BitLocker encryption, automatic updates via WSUS/Intune
  • Software deployment via GPO or Microsoft Intune

4. Access & Permissions

Users and groups are centrally managed via Active Directory. The organizational structure is reflected in Organizational Units (OUs) such as Sales, Marketing, IT, and Management. Each department has its own group policies and access rules.

OU Structure Example:

  • OU=Company
  • ├── OU=Sales
  • ├── OU=Marketing
  • ├── OU=IT
  • └── OU=Management

Permission Management:

  • Shared folders mapped via GPO per department
  • Access based on security groups (e.g., SG_Sales_Read, SG_IT_Admin)
  • GPOs for desktop restrictions, software policies, login scripts

5. Backup & Cybersecurity

  • Backup strategy: 3-2-1 rule (3 copies, 2 media types, 1 off-site)
  • Use of Synology NAS + encrypted cloud storage (e.g., Wasabi, Azure)
  • Automated daily and weekly backup schedules
  • Firewall with DPI, intrusion detection, and threat alerts
  • Company-wide MFA, strong password policies, regular security training
  • BitLocker for all endpoints, secure BIOS/UEFI settings

6. Support & Monitoring

  • IT helpdesk: GLPI or Spiceworks for ticket management
  • Remote access with AnyDesk or Microsoft Remote Desktop
  • Monitoring: PRTG Network Monitor for network/server performance
  • Log collection: Windows Event Viewer + centralized syslog
  • Automated alerts for CPU, disk, memory, and uptime issues

7. Demo Environment

A fully functional demo lab has been set up on a Mac M1 using Parallels Desktop with Windows Server 2022. It includes:

  • AD DS installation and domain configuration
  • Sample OUs and user accounts
  • GPOs for desktop wallpaper and password policy
  • Optional: Windows 11 VM joined to the domain
