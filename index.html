IT Infrastructure Blueprint

Tailored for a 20-Employee Tech Startup

1. Project Overview

This document outlines a complete IT infrastructure concept for a startup with 20 employees, covering network architecture, server environment, user and device management, security, backup strategies, monitoring, and cloud integration. The setup also includes a demo lab to showcase technical capabilities.

2. Network & Infrastructure

2.1 Hardware Recommendations

  • Router/Firewall: UniFi Dream Machine Pro – Unified network management, VPN, DPI, security gateway
  • Switch: Netgear 24-Port PoE+ Smart Managed Switch – VLAN capable, PoE for Access Points
  • Access Points: UniFi 6 Lite – Enterprise Wi-Fi 6 with seamless roaming
  • Server: Dell PowerEdge with Windows Server 2022 – Active Directory, DNS, DHCP, file services
  • NAS: Synology DS920+ – Local backup, file storage with RAID protection
  • UPS: APC Smart-UPS – Backup power for critical systems

2.2 Network Design & VLANs

  • VLAN 10 – Office LAN: Employee PCs, laptops
  • VLAN 20 – Servers: Server + NAS
  • VLAN 30 – Printers & VoIP phones
  • VLAN 40 – Guest Wi-Fi: Internet access only

2.3 IP Addressing & DHCP

Each VLAN uses a separate subnet (e.g., 192.168.10.0/24). DHCP is managed by the Windows Server for internal devices and by the router for guests.

2.4 Wireless Configuration

  • SSID 1: Startup-Office – WPA3, VLAN 10
  • SSID 2: Startup-Guest – VLAN 40, isolated

3. Client & Device Management

3.1 Standards & Software

  • Windows 11 Pro on all workstations
  • Standard apps: Microsoft 365, Zoom, PDF Reader, browser

3.2 Updates & Endpoint Security

  • Windows updates managed via WSUS
  • BitLocker enforced via GPO
  • Device lockdown: BIOS protection, USB port control

3.3 Software Deployment

  • GPO-based MSI installations
  • PowerShell scripts for onboarding
  • Optional: Intune integration for mobile/hybrid users

4. Access & Permissions

Centralized user and group management with Active Directory, structured by department:

OU=Company
├── OU=Sales
├── OU=Marketing
├── OU=IT
└── OU=Management
  • Role-based access with Security Groups (e.g., SG_Marketing_RW)
  • Shared drives and printer mapping via GPO
  • Login scripts for drive mapping and settings

5. Backup & Cybersecurity

  • 3-2-1 backup strategy: NAS + encrypted cloud + offline copy
  • Daily differential + weekly full backups
  • Firewall with intrusion detection and DPI
  • MFA for Microsoft 365 and VPN
  • User training: phishing simulations, password policy enforcement

6. Support & Monitoring

  • Helpdesk: GLPI or Spiceworks
  • Remote support: AnyDesk or RDP
  • Monitoring: PRTG Network Monitor
  • Alert rules: CPU > 90%, Disk space < 20%, Service failures

7. Demo Lab Setup

  • Parallels VM with Windows Server 2022
  • Active Directory with OUs and GPOs
  • Client VM (Windows 11) joined to domain
  • Test GPOs for desktop config and security

8. Cloud & Remote Work

  • Microsoft 365: Teams, OneDrive, Outlook
  • Azure AD Sync from local AD
  • VPN via UniFi Gateway
  • Mobile Device Management with Intune
  • Cloud backup via Synology C2 / Azure Backup

9. Summary & Outlook

This infrastructure provides a secure, centralized, and scalable setup for a modern startup. Future additions could include VoIP rollout, hybrid cloud architecture, automated onboarding, and zero-trust network enhancements.
