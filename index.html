IT Infrastructure Blueprint

for a Startup with 20 Employees

1. Project Overview

This virtual IT infrastructure test environment simulates a complete startup network setup using Windows Server 2022, Active Directory, DNS/DHCP, VLANs, and Group Policies. It is built on a Mac M1 using Parallels Desktop to demonstrate technical understanding and practical implementation.

7. Demo Environment

A fully functional demo lab has been set up on a Mac M1 using Parallels Desktop with Windows Server 2022. It includes:

  • AD DS installation and domain configuration
  • Sample OUs and user accounts
  • GPOs for desktop wallpaper and password policy
  • Optional: Windows 11 VM joined to the domain

Screenshots and videos available upon request or during presentation.

8. Cloud & Remote Work

As a modern startup, integrating cloud-based services and remote work tools enhances flexibility, scalability, and collaboration. The infrastructure supports hybrid and remote teams using Microsoft 365 and optional Azure services.

  • Microsoft 365: Includes Teams, OneDrive, Exchange Online for collaboration
  • Azure AD Sync: Option to extend local AD to Azure Active Directory
  • Conditional Access Policies for secure remote access
  • Mobile Device Management (MDM) with Microsoft Intune
  • Cloud backups for critical business data (e.g., Synology C2, Azure Backup)
  • VPN access configured via UniFi Gateway with user-based access control

9. Summary & Outlook

This IT infrastructure blueprint provides a scalable, secure, and modern foundation for a startup with 20 employees. With structured network segmentation, centralized management, proactive monitoring, and cloud integration, the company is well-prepared for growth, remote work, and operational continuity.

Future expansions could include advanced automation (e.g., scripting, onboarding workflows), expanded Azure services, VoIP integration, and further security hardening.
