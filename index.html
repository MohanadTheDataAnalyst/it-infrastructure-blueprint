Comprehensive IT Infrastructure Blueprint

Tailored for a 20-Employee Tech Startup

1. Project Overview

This document outlines a comprehensive IT infrastructure for a startup with 20 employees, including network architecture, physical hardware, virtual infrastructure, cloud integration, device lifecycle management, user access control, security hardening, support, monitoring, and disaster recovery.

2. Network & Infrastructure

2.1 Hardware Recommendations

  • Router/Firewall: UniFi Dream Machine Pro – Integrated network controller, firewall, DPI, VPN
  • Switch: Netgear GS728TP – 24-port managed PoE+ switch with VLAN and link aggregation
  • Access Points: UniFi 6 LR – Long-range Wi-Fi 6 for strong coverage
  • Server: Dell PowerEdge R550 with RAID 5, 128 GB RAM, dual Xeon
  • NAS: Synology RS1221+ – Central backup with Btrfs snapshots and cloud sync
  • UPS: APC Smart-UPS with SNMP monitoring module
  • Patch Panel + Rack: 12U rack cabinet for central structured cabling

2.2 Network Design

  • VLAN 10 – Employee LAN: Workstations, laptops
  • VLAN 20 – Servers: AD, File Server, DNS/DHCP
  • VLAN 30 – Printers & VoIP phones
  • VLAN 40 – Guest Wi-Fi: Isolated internet access
  • VLAN 50 – Management: Admin access and monitoring tools

2.3 IP Addressing

Private addressing scheme with static IPs for infrastructure, DHCP ranges per VLAN, reverse DNS zones configured.

2.4 DNS & DHCP

  • Primary DNS/DHCP via Windows Server 2022
  • Redundant DHCP failover scope
  • Conditional forwarders and split-brain DNS

2.5 Cabling & Wi-Fi

  • Cat6 cabling to every desk with wall-mounted faceplates
  • Ceiling-mounted APs with PoE
  • SSID Startup-Office (WPA3, VLAN 10), Startup-Guest (VLAN 40)

3. Virtualization & Server Roles

  • Hypervisor: VMware ESXi or Hyper-V Core
  • VM1 – Domain Controller + DNS/DHCP
  • VM2 – File Server with quota management and shadow copies
  • VM3 – Monitoring/Logging Server (Graylog, PRTG)
  • VM4 – Remote Desktop Gateway (optional)

4. Client Lifecycle Management

  • Standard Image Deployment via MDT or Autopilot
  • GPO hardening: USB blocking, BitLocker enforcement, AppLocker
  • Intune enrollment for remote/hybrid devices
  • Remote wipe for lost/stolen devices
  • Software: M365, Zoom, Notepad++, VPN Client

5. Identity & Access Management

  • Organizational Units per department
  • Group-based access control (least privilege)
  • Login script for drive/printer mapping
  • Azure AD Hybrid Join for cloud access
  • MFA with conditional access (geo, risk, device)

6. Backup, Security & Compliance

  • Veeam Backup to local NAS + encrypted cloud (Backblaze B2 or Azure)
  • Backup test restores monthly (sample file + full system)
  • Firewall rules per VLAN, IDS/IPS enabled
  • Security awareness training quarterly
  • Compliance: GDPR checklist, audit trails via Sysmon

7. Monitoring & IT Support

  • PRTG for uptime, SNMP, traffic, AD/SQL checks
  • Graylog or ELK for log aggregation
  • Ticketing via GLPI or Freshdesk
  • Self-service password reset portal
  • IT Wiki using BookStack or Confluence

8. Cloud & Remote Work

  • Microsoft 365 E3 Plan for all users
  • Azure AD + Intune + Defender for Endpoint
  • VPN access with MFA (WireGuard or L2TP/IPSec)
  • File access via OneDrive and mapped SharePoint drives

9. Demo Lab Environment

  • Parallels Desktop on Mac M1 with nested virtualization
  • 1x DC VM (Windows Server 2022), 1x Client VM (Win 11)
  • Test OU, GPOs: Drive mapping, desktop lock, AppLocker
  • Simulation of login flow, password reset, Intune registration

10. Summary & Outlook

This IT blueprint ensures security, scalability, compliance, and flexibility for a modern, cloud-integrated startup. It sets the foundation for DevOps, automation, and high-availability in the future.
